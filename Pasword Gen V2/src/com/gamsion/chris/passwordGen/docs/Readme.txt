Readme:

Usage:
This could be used for classes/groups so that you could mass produce password
keys for members of said group.

Instructions:
1. Type into the first box the number of passwords you want generated.
2. Type into the second box the number of uppercase letters you want in the passwords.
3. Type into the third box the number of lowercase letters you want in the passwords.
4. Type into the fourth box the number of numbers you want in the passwords.
5. Click "Generate."
6. ???
7. Your passwords will be presented in a dialogue. If there are too many dialogues to
display, you could find a text file that was outputed in a folder within the same
directory as the jar called "PasswordGenerator/[date formula].txt"
